# Cadence Sublime Text Package

Production-ready Sublime package scaffold for Cadence.

## Setup

1. Copy this folder into Sublime Text `Packages/Cadence`.
2. Open `Preferences > Package Settings > Cadence`.
3. Set `api_key` and `endpoint`.
4. Run `Cadence: Test Connection`.

Heartbeats are debounced and use `ide = "sublime"`.
